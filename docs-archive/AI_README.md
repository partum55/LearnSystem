## 🎯 Що можна генерувати

### 1. Повні курси
**Де**: Courses → 🤖 AI Generator  
**Що**: Курс з модулями, завданнями, квізами

### 2. Модулі для курсу
**Де**: Course Detail → Modules → 🤖 AI Генерація  
**Що**: 1-12 модулів з описами

### 3. Завдання для модуля
**Де**: Course Detail → Module → 🤖 AI Завдання  
**Що**: Практичні завдання різних типів

## 🔑 API ключі

### Groq API (Хмарний) - РЕКОМЕНДОВАНО ⭐
```bash
# 1. Отримати безкоштовний ключ на https://console.groq.com/
# 2. Додати у .env:
LLAMA_API_URL=https://api.groq.com/openai/v1
LLAMA_API_KEY=gsk_ваш_ключ_тут
LLAMA_MODEL=llama-3.1-70b-versatile

# 3. Запустити
./start.sh --with-ai
```

### Альтернативні провайдери
- **Together AI**: https://api.together.xyz/
- **OpenRouter**: https://openrouter.ai/
- **Ollama** (локально, якщо хочете): встановити модель

**Детально**: `LLAMA_API_SETUP.md`
