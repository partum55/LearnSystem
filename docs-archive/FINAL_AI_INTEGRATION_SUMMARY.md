## 📋 Що було виконано

### 1. ✅ Куди вставляти Llama API

#### Ollama (Локально) - РЕКОМЕНДОВАНО
**API ключ НЕ потрібен!** Ollama працює локально без хмарних сервісів.

**Налаштування** (вже зроблено):
```bash
# .env
LLAMA_API_URL=http://localhost:11434
LLAMA_MODEL=llama3.1
# Без API ключа!
```

**Запуск**:
```bash
# Встановити Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Завантажити модель
ollama pull llama3.1

# Запустити все разом
./start.sh --with-ai
```

#### Зовнішні API (якщо потрібен хмарний сервіс)

**Варіанти провайдерів**:
- **Groq** (швидкий, безкоштовний tier): https://console.groq.com/
- **Together AI** (хороші ціни): https://api.together.xyz/
- **OpenRouter** (багато моделей): https://openrouter.ai/

**Налаштування з API ключем**:
```bash
# .env
LLAMA_API_URL=https://api.groq.com/openai/v1
LLAMA_API_KEY=gsk_ваш_ключ_тут
LLAMA_MODEL=llama-3.1-70b-versatile
```

**Детальна інструкція**: `LLAMA_API_SETUP.md`

### 2. ✅ Генерація окремих елементів (модулі, завдання, квізи)

Створено компонент `AIElementGenerator` який підтримує:

#### 📚 Модулі
- Генерація для існуючого курсу
- Налаштування кількості (1-12)
- Тривалість у тижнях

#### 📝 Завдання (Assignments)
- Генерація для конкретного модуля
- Типи подання: FILE, TEXT, URL, CODE
- Налаштування балів (1-1000)
- Кількість завдань (1-10)

#### ❓ Квізи
- Генерація питань (5-50)
- Складність: EASY, MEDIUM, HARD
- Ліміт часу (5-180 хв)

---

### 3. ✅ Можливість задавати контекст

#### Автоматичний контекст
Система автоматично передає:
- **Для модулів**: код курсу, назва, опис
- **Для завдань**: інформація про курс + модуль
- **Для квізів**: тема модуля

#### Додатковий контекст
Користувач може додати:
- Рівень студентів
- Попередні знання
- Особливі вимоги
- Конкретні теми

## 🎨 Як використовувати у UI

### 1. Генерація повного курсу
**Де**: Courses → кнопка "🤖 AI Generator"

**Що робить**:
- Створює курс з нуля
- Генерує модулі
- Додає завдання
- Створює квізи

### 2. Генерація модулів
**Де**: Course Detail → вкладка Modules → кнопка "🤖 AI Генерація"

**Контекст передається автоматично**:
```
Курс: CS101: Intro to Programming
Опис: Learn Python basics for beginners
```

**Додатковий контекст** (можна вказати):
```
Студенти 1 курсу, базові знання програмування
Фокус на практичні завдання
```

### 3. Генерація завдань у модулі
**Де**: Course Detail → розгорнути модуль → "🤖 AI Завдання"

**Автоматичний контекст**:
```
Курс: CS101: Intro to Programming
Модуль: Variables and Data Types
Опис: Learn about different data types in Python
```

**Параметри які можна налаштувати**:
- Кількість завдань (1-10)
- Тип подання (FILE, TEXT, URL, CODE)
- Максимальний бал (1-1000)
- Мова (українська/англійська)

**Приклад промпту**:
```
Створи 3 практичні завдання на роботу зі змінними:
1. Легке - створення змінних різних типів
2. Середнє - конвертація типів
3. Складне - робота зі складними структурами даних
```

## 💻 Приклади використання API

### Генерація модулів з контекстом

```typescript
import { aiApi } from '../api/ai';

// Базова генерація
const result = await aiApi.generateModules({
  courseId: "course-uuid",
  prompt: "Створи 4 модулі про Python",
  language: "uk",
  moduleCount: 4
});

// З додатковим контекстом
const resultWithContext = await aiApi.generateModules({
  courseId: "course-uuid",
  prompt: "Створи поглиблені модулі про ООП",
  language: "uk",
  moduleCount: 3,
  context: "Студенти 2 курсу, знають основи Python",
  durationWeeks: 2 // Кожен модуль 2 тижні
});
```

### Генерація завдань з контекстом

```typescript
// З повним контекстом
const assignments = await aiApi.generateAssignments({
  moduleId: "module-uuid",
  moduleTopic: "Робота з файлами в Python",
  language: "uk",
  assignmentCount: 3,
  context: "Студенти 1 курс, базовий рівень",
  maxScore: 100,
  submissionType: "CODE"
});
```

### Генерація квізів

```typescript
const quiz = await aiApi.generateQuiz({
  moduleId: "module-uuid",
  topic: "Декоратори в Python",
  language: "uk",
  questionCount: 10,
  context: "Студенти знають основи функцій",
  timeLimit: 30,
  difficulty: "MEDIUM"
});
```

```typescript
// Поганий приклад - мало контексту

context: `
Студенти: 1 курс, вивчили базовий Python
Вже знають: змінні, цикли, умови
Нові теми: def, параметри, return
Рівень: початковий до середнього
`
```

## 🎓 Приклади реальних сценаріїв

### Сценарій 1: Новий курс з Python

```typescript
// 1. Генерація структури
const course = await aiApi.generateCourse({
  prompt: "Курс Python для Data Science",
  language: "uk",
  include_modules: true,
  module_count: 8
});

// 2. Додавання завдань до кожного модуля
for (const module of course.data.modules) {
  await aiApi.generateAssignments({
    moduleId: module.id,
    moduleTopic: module.title,
    context: `Модуль: ${module.description}`,
    assignmentCount: 3,
    maxScore: 100
  });
}
```

```typescript
Студенти вже знають базовий Docker
```
